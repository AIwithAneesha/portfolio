    Conversational Chatbot

    The frontend of the app is streamlit

    The backend of the app is built using Python and Langchain. We use chains, few shot template of langchain.

    The LLM Models used are Hugging face API and Openai api.The other free APIs used are Google calender API to set the reminder and Weather API.

    Text is classified using OpenAI Chain(create_tagging_chain) into mental,therapy ,anxious,depressed,hurt,stress,'addiction', weather related,Remainder,Play a video,play a game,Play a music,Tell a joke, general knowledge question

    1.If classified into mental,therapy ,anxious,depressed,hurt,stress,addiction, using the LLMChain and Huggingface API

    2.If weather related then weather API(Free is used)

    3.If remainder, the date is segmented into start and end datetime then added to Google Calender

    RAG used with Pinecone for context management

    You can find the source code for the Conversational Chatbot on GitHub: https://github.com/AIwithAneesha/Conversational_chatbot
